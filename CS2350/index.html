HTML

HTML stands for "Hyper Text Markup Language". This is a tag-bases language where tags are created with greater than and less than signs.

For Example

<p> is a paragraph </p>

There are special characters such as &lt; that is a less-than sign.

Inline vs Block

Inline means that is inline with the text. Block level forces a new line.

Block Level: div, p, h1, h2, h3, h4, h5, h6, li,

Inline Elements are span, img, a , strong, em
